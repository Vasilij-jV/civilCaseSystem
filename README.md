# Civil Case System

Локальная Django-система для систематизации материалов гражданского дела:
- документы
- доказательства
- версии иска
- замечания суда
- правовые нормы
- хронология событий

## Стек
- Python
- Django
- SQLite

## Структура проекта
- `src/` — Django-проект
- `requirements.txt` — зависимости
- `.gitignore` — исключения для Git
- `README.md` — описание проекта

## Цель
Создать удобную систему навигации по материалам дела с возможностью:
- хранить документы и доказательства
- связывать документы с правовыми нормами
- отслеживать замечания суда
- безопасно развивать структуру проекта через Git и миграции Django
